## 📌 About

This repository contains a Spring Boot learning project created while following the **Spring Boot – Mastering the Fundamentals** course by Code With Mosh.

The purpose of this project is to understand core Spring Boot concepts and build a strong foundation.

---

## 📚 Topics Covered

- Spring Boot project setup  
- Dependency Injection  
- Configuration using application.properties  
- Spring Data JPA basics  
- Entities and Repositories  
- Database integration  

---

## 🛠 Tech Stack

- Java  
- Spring Boot  
- Spring Data JPA  
- Maven  
- MySQL  

---

## 🎯 Goal

Build strong fundamentals in Spring Boot before moving to Web MVC and REST APIs.
